## Projeto Proposta : O projeto consiste no desenvolvimento de um aplicativo inovador para a divulgação de eventos. A plataforma permitirá que organizadores promovam seus eventos de forma eficiente, alcançando um público maior. Além disso, os usuários poderão explorar uma variedade de eventos, filtrar por categorias e localizações, receber notificações personalizadas e até interagir com os organizadores. O objetivo é criar uma experiência intuitiva e acessível para conectar pessoas a eventos de seu interesse, tornando a descoberta e participação mais dinâmica e envolvente.
